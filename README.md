# 🧮 계산기 프로그램

## 📌 프로젝트 소개
- 간단한 사칙연산 계산기 프로그램
- Java 기반으로 구현
- 제네릭, 스트림, 람다, 예외처리 활용

---

## ⚙️ 주요 기능

### 1. 사칙연산
- +, -, *, / 연산 지원

### 2. 결과 저장
- 연산 결과를 리스트에 저장

### 3. 결과 삭제
- 가장 오래된 결과 삭제

### 4. 특정 값보다 큰 결과 출력
- 스트림과 람다를 사용하여 필터링

---

## 💡 느낀점
- 제네릭을 처음 사용해보며 타입 유연성을 이해했다
- 스트림을 통해 반복문 없이 데이터 처리가 가능했다
